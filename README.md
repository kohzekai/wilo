# Vercel Simple Deployment (No Database)

## Quick Deploy - 5 Minutes!

This folder contains everything you need to deploy to Vercel.

### What's Inside:
- `index.html` - Your pump selection optimizer (enhanced version)
- `vercel.json` - Vercel configuration

### Deployment Steps:

#### Option 1: Drag and Drop (Easiest)
1. Go to https://vercel.com/new
2. Drag this entire folder into the upload area
3. Click "Deploy"
4. Done! Your site is live at: `yourproject.vercel.app`

#### Option 2: Vercel CLI
```bash
# Install Vercel CLI
npm install -g vercel

# Navigate to this folder
cd vercel-simple

# Deploy
vercel

# Follow prompts, then your site is live!
```
